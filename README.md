# Widget Hider v2.0

Расширение для SillyTavern, позволяющее скрывать плавающие виджеты других расширений. 
Полная поддержка мобильных устройств!

## ✨ Возможности

### 📱 Мобильная версия
- **Полная адаптация** под любой размер экрана (телефон, планшет, ПК)
- **Touch-события** для выбора виджетов на сенсорных устройствах
- **Центрированные модальные окна** для удобства на всех устройствах

### 🎯 Выбор виджетов
- **Одиночный выбор** — нажмите на виджет для добавления
- **Множественный выбор** — выберите несколько виджетов за раз
- **Авто-сканирование** — найдёт все плавающие элементы автоматически

### 📊 Счётчики
- Показывает количество виджетов в списке
- Показывает сколько виджетов сейчас скрыто (красный бейдж)
- Счётчик виден прямо в меню расширений ST

### 🎨 Интерфейс
- Центрированное меню и панели
- Тёмная тема в стиле таверны
- Плавные анимации
- Backdrop для модальных окон

## 📖 Как использовать

### Открыть меню
1. Нажмите на кнопку расширений (иконка волшебной палочки ✨)
2. Найдите пункт **Widget Hider**
3. Откроется центрированное меню с опциями

### Добавить виджеты
**Способ 1: Одиночный выбор**
- Выберите «Выбрать виджет»
- Нажмите/тапните на нужный виджет
- Он добавится в список

**Способ 2: Множественный выбор**
- Выберите «Выбрать несколько»
- Нажимайте на виджеты (они подсветятся)
- Нажмите повторно для отмены выбора
- Подтвердите добавление

**Способ 3: Авто-сканирование**
- Выберите «Авто-сканирование»
- Отметьте нужные виджеты
- Подтвердите добавление

### Скрыть/Показать
- Нажмите «Скрыть виджеты» / «Показать виджеты»
- Все виджеты из списка скроются/покажутся

### Управление списком
- Выберите «Управление списком»
- Нажмите на строку для подсветки виджета на экране
- Нажмите ✕ для удаления из списка

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Escape` | Закрыть меню / Отменить выбор |

## 🔧 Технические детали

- Сохраняет состояние в localStorage
- Не трогает системные элементы ST
- Z-index оптимизирован для корректного отображения
